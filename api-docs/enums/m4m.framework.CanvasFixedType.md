# Enumeration: CanvasFixedType

[m4m](../modules/m4m.md).[framework](../modules/m4m.framework.md).CanvasFixedType

**`language`** zh_CN

**`classdesc`**
设定画布的渲染大小，选择长或者宽作为标准锁定画布大小进行渲染。横屏选择FixedWidthType，竖屏选择FixedHeightType。目的是锁定屏幕大小，防止分辨率过高导致渲染压力过大

**`version`** m4m 1.0
